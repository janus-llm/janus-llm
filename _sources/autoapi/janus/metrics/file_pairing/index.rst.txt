.. py:function:: register_pairing_method(name = None)

   Registers a pairing method for pairing strings between files

   :param name: The name of the pairing method. If None, the function name is used.
   :param help: The help text for the pairing method.

   :returns: The decorator function.


.. py:function:: pair_by_file(target, reference, **kwargs)

   :param target: The target file text.
   :param reference: The reference file text.
   :param state: The current evaluation state.

   :returns: A list of tuples of the target and reference file text.

   :param state: The current evaluation state.

   :returns: A list of tuples of the target and reference file text.


.. py:function:: pair_by_line_comment(target, reference, **kwargs)

   Pairs the comments of a file together by line

   **WARNING**: Do not use, as this method is extremely brittle.

   :param target: The target file text.
   :param reference: The reference/reference file text.
   :param state: The current evaluation state.

   :returns: A list of tuples of the target and reference file text.
