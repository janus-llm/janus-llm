.. py:function:: bleu(target, reference, smooth_method = 'exp', smooth_value = None, lowercase = False, use_effective_order = True, **kwargs)

   Computes BLEU score using sacrebleu

   :param target: The target text.
   :param reference: The reference text.
   :param smooth_method: smoothing method to use.
   :param smooth_value: smoothing value (only for floor and add-k).
   :param lowercase: whether to lowercase the data.
   :param use_effective_order: Don't use n-gram orders without matches.

   :returns: The BLEU score float.
